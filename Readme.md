# Projekt TPM

## Opis projektu
Układ ma na celu zapalanie diod od zielonych do czerwonych, w zależności od amplitudy drgań. Im większa jest amplituda drgań tym więcej diod się zaświeci. Do budowy tego układu użyje:

* arduino uno

* Mikrofon

* 9 diod LED

* 9 rezystorów 220 ohm

* przewody

* płytka stykowa
